- nextmove-case-study.html      Complete NextMove case study with all sections
- case-study.html               Template for additional case studies
- styles-light.css              Main stylesheet (light neutral theme)
- case-study-light.css          Case study styles + bubble navigation

- README.txt                    This file

✅ Section jump navigation (6 bubbles: Empathize → Test + Reflection)

HOW TO USE LOCALLY:
1. Extract all files to a folder
2. Open index.html in your web browser
3. Click "NextMove" card to view the complete case study
4. See the 6-bubble navigation below the case study overview

CUSTOMIZATION:

Add your photo (homepage):
- In index.html, find "hero-photo-placeholder" div (around line 60)
- Replace with: <img src="images/your-photo.jpg" alt="Quinn Boyle">
- Recommended size: 400x400px

Add project images:
- Create an "images" folder in the same directory
- Save your images there
- Replace placeholder divs with: <img src="images/filename.jpg" alt="Description">

Update NextMove case study images:
- In nextmove-case-study.html, find "case-image-placeholder" divs
- Replace with your actual screenshots from Figma
- Recommended sizes:
  * Hero image: 1600x900px
  * Section images: 1200x800px
  * Component/persona images: as needed

Create additional case studies:
- Duplicate case-study.html
- Rename to: case-study-project2.html, case-study-project3.html
- Update project cards in index.html to link to these files
- Fill in all sections following the NextMove example

DEPLOYMENT (FREE OPTIONS):
